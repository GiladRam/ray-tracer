# `ray-tracer`

## Introduction

`ray-tracer` is a 3D rendering engine written in C++.

## Features
* Phong Model and Monte Carlo Path Tracing Model
* rendering of plane, sphere and mesh
* read mesh from a simple obj file
* mesh smoothing
* read scene from a simple json file
